# Финальный проект по работе с API, формами и публикацией на GitHub Pages.

Сайт доступен по ссылке: 
**https://yoursimagination.github.io/mesto-deployment/**

# Установка и запуск проекта локально
# Установка зависимостей
npm install

# Сборка проекта
npm run build

# Запуск dev-сервера
npm run dev

# Список технологий
1) HTML
2) CSS
3) JS
4) Vite + Live Server

Проект включает в себя:
- Работу с API (получение/добавление/удаление карточек)
- Редактирование профиля и аватара
- Возможность поставить/убрать лайк
- Валидацию форм
- Автоматический деплой на GitHub Pages

# Внимание
Токен автора защищён от копирования, возможно скоро будет добавлено больше методов для защиты всей работы

# Публикация проекта

Проект публикуется автоматически при push в ветку `main`:
```
git add .
git commit -m " изменения"
git push origin main
```
